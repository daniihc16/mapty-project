# mapty-project
This is a project from the Udemy course "The Complete JavaScript Course 2023: From Zero to Expert!" by Jonas Schmedtmann.

## Project Description
This project is a workout tracker that uses the Leaflet library to display a map. The user can click on the map to add a workout. The user can choose between two types of workouts: running and cycling. The user can also choose the distance, duration, cadence, and elevation gain of the workout.

## Features
* The user can add a workout by clicking on the map.
* The user can choose between two types of workouts: running and cycling.
* The user can choose the distance, duration, cadence, and elevation gain of the workout.
* The user can see all of their workouts displayed on the map.
* The user can see all of their workouts displayed in a list.
* The user can focus a workout on the map by clicking on it in the list.

## Installation and Setup Instructions
Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Start Server:

`npm start`

To Visit App:

`localhost:1234`
